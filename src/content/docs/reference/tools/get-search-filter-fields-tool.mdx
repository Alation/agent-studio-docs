# Get Search Filter Fields Tool

This tool finds the fields you can filter on when searching the catalog. It scans all available filters and returns the ones most semantically similar to your search term, so an agent can discover useful filters before applying them.

## When to use it

Use this as the first step in a filtered search: find a relevant field here, then look up its values with the [Get Search Filter Values Tool](/agent-studio-docs/reference/tools/get-search-filter-values-tool), then pass the filter to the [Search Catalog Tool](/agent-studio-docs/reference/tools/search-catalog-tool).

## Parameters

| Parameter | Required | Type | Suggested scope | What it does |
| --- | --- | --- | --- | --- |
| `search_term` | Yes | string | Agent determined | The topic to find relevant filter fields for. |
| `limit` | No | integer (default `10`) | Fixed | Maximum number of fields to return. |

<Aside type="note">
See [Configuring tool parameters](/agent-studio-docs/reference/tools/introduction/#configuring-tool-parameters) for what the scope column means.
</Aside>

## Example usage

```jsonc
{
  "search_term": "compliance",
  "limit": 10
}
```
