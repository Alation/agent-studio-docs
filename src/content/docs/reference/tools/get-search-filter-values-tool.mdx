# Get Search Filter Values Tool

This tool can be used to find particular values for a given field, which can be used to filter results from searching the Alation Data Catalog.
Use the search term to help narrow down results by semantically searching.
To search for fields which can be used for filtering, use the `get_search_filter_fields` tool.

## Example usage

```json
{
  "field_id": 0,
  "search_term": "governance",
  "limit": 10
}  
```
