# List Data Products Tool

This tool can be used to fetch multiple data products based on a natural language query.

## Example usage

```jsonc
{
  "search_term": "Human Resources",
  "limit": 5,
  "marketplace_id": "my-marketplace"  // Optional
}
   
```
