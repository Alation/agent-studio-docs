**Release Date:** 2026-06-17

**Version Range:** v26.143.1 -> v26.153.1

## Highlights

- New tools make it faster to build, populate, and validate the feeds that power your agents.
- Agents now improve themselves over time by applying reviewer suggestions and prioritizing the data points that need attention.
- Chat, task handling, and error messages are more reliable across the board.

## What's New

### Features

- **Faster feed building:** A new set of guided tools helps you create, populate, and check feeds with less manual effort. You can build a new feed from scratch or start from one of your saved feeds.
- **Agents that improve over time:** Agents can now automatically apply reviewer suggestions, keep better track of context as they work, and focus on the data points that most need attention.
- **Agent health at a glance:** You can now see an agent's health directly in the troubleshooting view, making it easier to spot and resolve issues.
- **Expanded agent capabilities:** Agents can handle more work per run, return richer connector details, and report their health during connector troubleshooting.
- **New insights skill:** A new skill surfaces query and usage insights, and more context-search capabilities are now available to agents.
- **Better usage analytics:** Activity is now attributed to individual users, giving you clearer visibility into how Agent Studio is used.

### Improvements

- **Cleaner organization:** Refined the underlying category structure for a more intuitive experience.

### Bug Fixes

- **Chat:** Your chat list now shows only your own chats across every role, and very large tool results no longer cause errors.
- **Agents:** Fixed an issue where some input values were not passed correctly to sub-agents.
- **Tasks:** Tasks are no longer incorrectly marked as failed during a restart, and interrupted tasks are recovered more reliably.
- **Error messaging:** Clearer messages for metric issues and for invalid document links.
- **Triggers:** Conversation context is now carried through correctly for all triggers.
- **Search:** Improved how catalog links are handled in search results.
- **Reliability:** Hardened the context API and fixed an authentication issue affecting certain redirect-based lookups.

### Breaking Changes

None in this release.

## Customer Impact

None in this release.
