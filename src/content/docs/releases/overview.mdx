## Release Schedule

Backend releases (`alation-ai`) are deployed weekly. Frontend releases (`alation-ui`) follow a separate cadence:

| Namespace | Release Frequency | Description |
|-----------|-------------------|-------------|
| `preview` | Weekly (with backend) | Bleeding-edge features for early adopters |
| `alation-default` | Bi-weekly | Stable production release |

## Versioning

Releases follow semantic versioning: `MAJOR.MINOR.PATCH`

- **MAJOR**: Breaking changes that require customer action
- **MINOR**: New features, backward-compatible
- **PATCH**: Bug fixes, backward-compatible
