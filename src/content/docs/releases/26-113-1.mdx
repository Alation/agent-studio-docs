**Release Date:** 2026-05-27

**Version Range:** 26.80.3 → 26.113.1

## Highlights

- Added Data Quality (DQ) standards recommendation capabilities to help enforce data governance policies.
- Introduced scheduled chats with the `run_chat` workflow tool for automated agent interactions.
- Enhanced Secure Cloud diagnostics with agent infrastructure health checks and knowledge base queries.

## What's New

### Features

- **Data Quality Standards:** New `fetch_dq_standards` tool enables agents to retrieve and recommend data quality standards from the DQ Manager, helping enforce governance policies.
- **Scheduled Chats:** You can now schedule automated chat sessions using the `run_chat` workflow tool for recurring agent interactions.
- **Secure Cloud Diagnostics:** Added agent infrastructure health diagnostics with active ping checks and decision matrix support.
- **Lineage Creation Skill:** New skill for creating lineage with discovery references.
- **Search Result Assets:** Added search result asset type for improved catalog search.
- **Navigate UI Capability:** Agents can now trigger UI navigation actions.
- **Asset Content Tool:** New `get_asset_content` tool for retrieving unstructured data from catalog assets.

### Improvements

- **Codemode Enhancements:** Migrated filter fields and values to skills architecture for better modularity.
- **Output Constraints:** Alamigo agent now supports output constraints in prompts for more controlled responses.
- **BI Object Handling:** Consolidated BI object type handling into `CatalogObjectType` for consistency.
- **Data Product Queries:** Data product query agent now accepts version parameters.

### Bug Fixes

- Fixed SMTP tool to treat empty string sensitivity labels as None.
- Fixed tool sync TTL cleanup to run on its own cadence.
- Fixed metadata listing to prefer listed versions and drop unlisted products from recent data products.
- Fixed data quality tool column type normalization and JSON decode error handling.
- Fixed various migration chain issues.

### Breaking Changes

None in this release.

## Customer Impact

None in this release.
