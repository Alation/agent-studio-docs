**Release Date:** 2026-05-27

## Highlights

- Introduced a new scheduling feature for agents, allowing more flexible and precise control over execution times.
- Enhanced performance for large-scale data processing, reducing execution times significantly.
- Resolved critical bugs to improve overall stability and reliability.

## What's New

### Features

- **Agent Scheduling:** You can now schedule agents to run at specific times or intervals directly within Agent Studio. This feature provides greater flexibility and control over your workflows.
- **Customizable Notifications:** Added the ability to configure notifications for agent execution statuses, so you can stay informed about successes, failures, or other key events.

### Improvements

- **Performance Enhancements:** Optimized the execution engine to handle larger datasets more efficiently, resulting in faster processing times for complex workflows.
- **UI Enhancements:** Improved the user interface for managing agent configurations, making it more intuitive and user-friendly.
- **Search Functionality:** Enhanced the search capabilities within Agent Studio, enabling quicker access to agents and workflows.

### Bug Fixes

- Fixed an issue where agent execution logs were not displaying correctly in certain cases.
- Resolved a problem that caused occasional timeouts when running multiple agents simultaneously.
- Addressed a bug where custom scripts were not saving properly under specific conditions.

### Breaking Changes

None in this release.
