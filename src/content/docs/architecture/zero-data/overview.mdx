In a Zero Data deployment, customer data never passes through Alation's cloud.
The client browser sends requests directly to a [k3s](https://k3s.io/)-based agent running in the customer's environment, and the agent communicates back to cloud services over an mTLS tunnel when it needs catalog metadata or session context.
