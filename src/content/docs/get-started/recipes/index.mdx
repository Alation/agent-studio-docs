This page maps common use cases to what you build and where you deliver it, with curated end-to-end walkthroughs for the patterns we see working at real enterprises.
[Build](/agent-studio-docs/build/overview/) covers the building blocks; [Integrations](/agent-studio-docs/integrations/overview/) covers the delivery channels.

## Use case map

| If you want to... | Build | Deliver via |
|-------------------|-------|-------------|
| Get alerted when a business condition fires | [Scheduled alert emails](/agent-studio-docs/get-started/recipes/scheduled-alert-emails/) | Email or Slack |
| Ask questions of your data in natural language | A [Query-style agent](/agent-studio-docs/build/agents/query/) with [SQL tools](/agent-studio-docs/build/tools/sql-execution-tool/) | [Web UI or Slack](/agent-studio-docs/integrations/slack/setup/) |
| Power your own app with chat-with-data | [Embedded chat via REST](/agent-studio-docs/get-started/recipes/embedded-chat-rest-api/) | [REST API](/agent-studio-docs/integrations/rest-api/auth/) |
| Bring trusted catalog context into Claude, ChatGPT, Copilot, or Cursor | Nothing — use the built-in tools | [MCP client setup](/agent-studio-docs/integrations/mcp-clients/claude/) |
| Clean up your catalog or BI estate in bulk | [Bulk trust flags](/agent-studio-docs/get-started/recipes/bulk-trust-flags/), [BI rationalization](/agent-studio-docs/get-started/recipes/bi-report-rationalization/) | Web UI, or [scheduled flows](/agent-studio-docs/build/schedules/scheduling-flows/) |
| Work with Alation from your AI coding tool | Nothing — install the plugin | [Plugins (Agent Skills)](/agent-studio-docs/integrations/plugins/overview/) |
